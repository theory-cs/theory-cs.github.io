Which of \(P_1\), \(P_2\), \(P_3\) has movie preferences most similar to \(P_4\)?

One approach to answer this question: use functions to define distance between user preferences.

Define the following functions whose inputs are ordered pairs of \(3\)-tuples each of whose components
comes from the set \(\{-1,0,1\}\)
\(\displaystyle d_{1}(~ (x_1, x_2, x_3) , (y_1, y_2, y_3) ~) = \sum_{i=1}^3\left( (\lvert~x_i-y_i~\rvert + 1) \textbf{ div } 2 \right)\) \(\displaystyle d_{2}(~ (x_1, x_2, x_3) , (y_1, y_2, y_3) ~) = \sqrt{ \sum_{i=1}^3 (x_i - y_i)^2}\)
\(d_1(P_4, P_1)\) \(d_1(P_4, P_2)\) \(d_1(P_4, P_3)\)
\(d_2(P_4, P_1)\) \(d_2(P_4, P_2)\) \(d_2(P_4, P_3)\)
